A popular JS library that designed to simplify DOM-oriented client-side scripting. Lightweigth, "write less, do more", JS library. jQuery creates higher-level methods needed for achieving common tasks by wrapping several JavaScript statements into a single call

Features

The jQuery library contains the following features:
• HTML/DOM manipulation
• CSS manipulation
• HTML event methods
• Effects and animations
• AJAX (Asynchronous JavaScript and XML)

Syntax

• Basic syntax is: $(selector).action()
• $ sign to specify jQuery
• selector to specify the HTML elements to select/query
• action is the action to be performed on the element(s)

• Use jQuery Selector to select/query HTML elements
• Actions can then be performed on the selected/queried elements
• jQuery functions can be stored in a separate .js file and referenced using the “script” element’s “src” attribute

Document Ready Event

• Document Ready Event is used to prevent any jQuery code from running before the document is finished loading
• Good practice to wait for the document to be fully loaded and ready before manipulating it
• Problems if JavaScript Statements run before document is fully loaded:

• Hide an element that is not created yet
• Get the size of an image that is not loaded yet

Selectors

jQuery selectors: select and manipulate HTML element(s).
jQuery selectors are used to “find" HTML elements based on their name, id, classes, types, attributes, and values of attributes

jQuery - Get and Set CSS Classes

jQuery has several methods for CSS manipulation.
• addClass(): Adds one or more classes to the selected elements
• removeClass(): Removes one or more classes from the selected elements
• toggleClass(): Toggles between adding/removing classes from the selected elements
• css(): Sets or returns one or more style properties for the selected elements
