What is JavaScript?

Use to make a static page dynamic.

  1. Get html file from the web server
  2. Send back html
  3. Execute any JS as required
  4. Browser can layout and display the page to the user

What Can JavaScript Do?

JavaScript can change HTML content.

JavaScript Can Change Images

JavaScript can change the style of an HTML element.

Where does JS go?

JS can be linked to an HTML page in a number of ways.

  • Inline
    - the practice of including JS code directly w/in certain HTML attributes (not rec cuz hard to maintain)
  • Embedded
    - the practice of place JS code w/in a script element
  • External
    - the JS code is in a seperate external file that html can access (access by .js)

JavaScript Syntax
