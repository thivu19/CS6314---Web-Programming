Where does JS go?

JS can be linked to an HTML page in a number of ways.

  • Inline
    - the practice of including JS code directly w/in certain HTML attributes (not rec cuz hard to maintain)
  • Embedded
    - the practice of place JS code w/in a script element
  • External
    - the JS code is in a seperate external file that html can access (access by .js)

JS Objects

Almost everything in the language is an obj. Obj in JS are prototype-based rather than class-based. Prototype base language: INheritance in which obj are inherited directly from other objs. It also does not use classes.

JS Obj Methods

JavaScript methods are actions that can be performed on objects

Using Built-In Methods

Arrays

  • The push() add an item to the end of an existing array
  • The pop() method can be used to remove the last element of an array
  • The valueOf() method returns an array as a sting
  • The join() method also joins all array elements into a string
  • The shift() method removes the first element of an array, and "shifts" all other elements one place down
  • The unshift() method adds a new element to an array (at the beginning)
  • The slice() method returns the selected elements in an array, as a new array obj
  • The splice() method adds/removes items to/from an array, and returns the removed item(s)
  • The reverse() method reverses the order of the elements in an array
  • The sort() method sorts the items of an array. Alphabetical & ascending order
  • The lastIndexOf() mthod searches the array for the specified item, and returns its position
